# polly
Chat with LLM using Pollinatations.ai API.

It works in a web browser. You can save the polly.html file to your 'Desktop' and run it locally in the browser.

Or use it directly from here - https://saemonzixel.github.io/polly/polly.html

You can also install it as an app on your Android smartphone. Download the APK from https://saemonzixel.github.io/polly/Polly-1.1.apk to your smartphone and install it manually.
